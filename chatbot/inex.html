// ==== The Chat Flow ==== //

const chatFlow = { start: { message: "Hi! Choose a question below:", options: [ "What is this website about?", "How do the cards work?", "What did you use JavaScript for?" ] }, ... };

üìå This is a JavaScript object that acts like a map for the chatbot. Each key (like "start", "What is this website about?") represents a point in the conversation. Each value has: message: What the bot says. options: What the user can click next.



// ==== Tracking the current question ==== //

function toggleChat() { const chat = document.getElementById("chatContainer"); chat.style.display = chat.style.display === "flex" ? "none" : "flex"; if (chat.style.display === "flex") { document.getElementById("chatBox").innerHTML = ""; showMessage("bot", chatFlow.start.message); showOptions(chatFlow.start.options); currentQuestion = "start"; } }

This function opens or closes the chat window. When it opens: It clears old messages. It starts the conversation fresh and It shows the first message and buttons.



// ==== When a question is cliked: sendQuestion() ==== //

function sendQuestion(question) { showMessage("user", question); if (chatFlow[question]) { const response = chatFlow[question]; showMessage("bot", response.message); showOptions(response.options); currentQuestion = question; } else { showMessage("bot", "Sorry, I don't understand that."); showOptions(chatFlow.start.options); } }

This function: Displays the user‚Äôs question in the chat.

*Looks up the bot‚Äôs response from the chatFlow object.

*Updates the screen with the next message and options.

*If the question isn‚Äôt recognized, it shows an error message.



// ==== Adding messages to the cat box ==== //

function showMessage(type, text) { const chatBox = document.getElementById("chatBox"); const message = document.createElement("p"); message.className = type; message.innerText = text; chatBox.appendChild(message); chatBox.scrollTop = chatBox.scrollHeight; }

This function: Adds a new message to the chat visually. type is either "user" or "bot" ‚Äî used for styling. It scrolls the chat to the bottom automatically.



// ==== Showing the clickable buttons ==== //

function showOptions(options) { const optionsContainer = document.querySelector(".chat-options"); optionsContainer.innerHTML = ""; options.forEach((opt) => { const btn = document.createElement("button"); btn.innerText = opt; btn.onclick = () => sendQuestion(opt); optionsContainer.appendChild(btn); }); }

This function: Clears the old buttons. Creates new buttons based on the available options from chatFlow Each button sends a new question when clicked.
